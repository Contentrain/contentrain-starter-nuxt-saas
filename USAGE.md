# Contentrain Nuxt SaaS on Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https%3A%2F%2Fgithub.com%2FContentrain%2Fcontentrain-starter-nuxt-saas)

## Quick start

```bash
pnpm install
pnpm build
```

## Deploy with Netlify CLI

From the repository root:

```bash
pnpm dlx netlify-cli init
```

If the repository is already connected to a Netlify project:

```bash
pnpm dlx netlify-cli link
```

## Build contract

- Build command: `pnpm deploy:netlify`
- Publish directory: `framework-managed`
- Node version: `22`
